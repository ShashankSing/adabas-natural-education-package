Introduction to Natural and Adabas programming

Natural is a programming language designed to simplify the implementation of business solutions. It is similiar to JAVA in that it is an interpreted language; during execution Natural source programs are dynamically "compiled" into a platform independent byte code (object) format which is then executed in a platform dependent environment.

Folowing Natural topics are discussed here...

  • Condtional constructs
  • Arrays
  • Retrieval from database

Adabas is short for an "Adabtable database"(?). Optimized for large-scle transaction processing, provides ultra-high-performance and reliable data processing for enterprise business transactions.

So, we can understand Adabas as a database present at the back-end and use Natural as a language to access/modify it.
